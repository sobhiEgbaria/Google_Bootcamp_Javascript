# You cna use the following:

- use Array methods
- use simple for loop
- use simple if else

__for__
```javascript
const names = ['John', 'Jane', 'Jack', 'Jill'];

for (let i = 0; i < names.length; i++) {
    console.log(i);
}
```

__if-else__
```javascript
const name = 'John';

if (name === 'John') {
    console.log('Hello John');
} else {
    console.log('Hello Stranger');
}
```
